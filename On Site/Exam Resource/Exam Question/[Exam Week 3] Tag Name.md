# Description
ให้รับข้อความเข้ามาหนึ่งของความ เเล้วให้เเสดงผลในลักษณะนี้
|------------|
|        Mei |
|------------|
ปล. ตัวอักษรมีเเค่ 1 - 10 เท่านั้น

# Specification
|Input|Output|
|-|-|
|มีหนึ่งบรรทัด คือ ข้อความ|เป็นลักษณะที่ให้เเสดงผลดังตัวอย่าง|

# Sample Input / Output
Input :
```
Mei
```

Output :
```
|------------|
|        Mei |
|------------|
```

```
Overwatch
```

